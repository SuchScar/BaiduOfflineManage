# BaiduOfflineManage
百度矢量离线地图（基于v4.5.2版本sdk）百度地图SDK的离线地图dem太简陋，由于最近公司项目需求，自己写一个百度离线地图下载管理Demo，直接下载项目即可使用
  
现在的离线地图直接获取是全球地图，cityID>20000的是他国地图，反之则是我国地图，自行判断
